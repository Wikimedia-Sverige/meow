About

About MEOW

MEOW is developed by Wikimedia Sverige as part of the Content Partnerships Hub initiative.

What is MEOW?

MEOW is a tool to explore for educational resources cataloged in Metabase, the Wikibase instance maintained by the Wikimedia movement for tracking open educational materials.

The catalog contains resources such as online courses, videos, guides, tutorials, slide decks, and articles, all freely available and relevant to Wikimedia-related learning.

Data is harvested from Metabase via SPARQL and refreshed regularly. Each resource links back to its Metabase item, where you can view full metadata and edit it if you have an account.

Contributing

How to contribute to Metabase

Add or improve a resource

Anyone with a Metabase account can add new resources or improve existing ones. Visit metabase.wikibase.cloud and log in or create an account.

What makes a good resource entry?

  • Description - a summary of what the resource covers.
  • Language - the language(s) the resource is available in.
  • Keywords - relevant topics such as Wikidata, GLAM, open data, or artificial intelligence.
  • Publisher - the organization or project that produced the resource.
  • Publication date - when the resource was created or last updated.
  • Links - where the resource can be accessed.

Use the Improve data filters

The Improve data section in the left panel lets you filter for resources that are missing specific fields.
